# GPExpansion v1.1.8

## New Features

### Teleport by Claim Name
- `/claim tp <name>` now resolves custom claim names in addition to numeric IDs
- `/claim tp playerName:claimName` targets another player's named claim
- Both require `griefprevention.claim.teleport.other` for others' claims
- Tab completion for `/claim tp` and `/claimtp` includes named claims alongside IDs

### GUI Improvements
- **Manage Trust icon** now shows the **claim owner's head** instead of the viewer's head
- View Children item in ClaimOptionsGUI now also accepts `{count}` placeholder (alongside `{children}`)

### Player Commands
- Added `claim.gui.flags` and `claim.gui.options` to the default `player-commands` list in `config.yml`, granting players access to `/claim flags` and `/claim options` subcommands
